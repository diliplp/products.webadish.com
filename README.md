# WebAdish Products Site

This folder is a lightweight static site intended for `products.webadish.com`.

## Current Structure

- `/` home page for the plugin brand
- `/products/autosheets-google-sheets/` first plugin landing page
- `/pricing/`
- `/docs/`
- `/support/`
- `/privacy-policy/`
- `/terms/`
- `/refund-policy/`

## How to Expand

For each new plugin:

1. Duplicate `/products/autosheets-google-sheets/` into a new product slug
2. Update homepage cards to link to the new plugin
3. Reuse shared docs, pricing, support, and legal pages where appropriate
4. Replace placeholder product copy with the final WordPress.org and Pro details

## Notes

- The first plugin page is now wired for `WebAdish AutoSheets for Google Sheets`.
- It uses a reusable IA so the site can grow into a proper multi-plugin product hub.
